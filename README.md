# Laboratório de Ataques em Active Directory (Top 10)

Este projeto demonstra **10 técnicas comuns usadas em Red Team e Pentest em ambientes Active Directory**.

Cada técnica contém:

- teoria
- configuração do laboratório
- exploração com comandos reais
- detecção
- mitigação

## Topologia do Laboratório

Máquinas sugeridas:

- Kali Linux (Atacante)
- Windows Server 2012 (Domain Controller)
- Windows 10 (Workstation 1)

Domínio:

dom.local

Rede exemplo:
